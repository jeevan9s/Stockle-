About Stockle

How it Works

Stockle is a free-to-use financial advisory tool that illustrates a graph when the user inputs the ticker symbol for their company of choice.

Stockle sources data from Yahoo Finance using a Python backend. The relevant data is then fetched and used to generate a line a graph using JavaScript. The framework and styling of the site is done with HTML and CSS.
